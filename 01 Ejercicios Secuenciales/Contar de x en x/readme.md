# Contar de x en x

Se desea enseñar a un niño a contar de **x** en **x**.

Dado un numero **n** determinar el anterior en **x** y siguiente en **x**.
Imprime el anterior, **n** y el siguiente.

## Entrada

Dos números enteros; **n,** **x**

## Salida

Tres números enteros:

**Ejemplo:**

| Entrada |Salida  |
|--|--|
| 10 5|5- 10- 15  |
| 100 3 |97- 100- 103  |
